# qqmusic-vue2.x
基于vue2.x  实现的一个音乐播放器，接口抓取自QQ音乐网页，安卓端
最开始只是想做一个音乐播放器，后来看到qq音乐9.0内测版外观不错，所以做了这个。前期没有规划好，有很多冗余代码，有时间会重构一下。

QQ音乐的API可能会改变，可能会出现无法使用的情况

# 如何运行
```javascript
git clone https://github.com/poporeki/qqmusic-vue2.x.git
cd qqmusic-vue2.x
npm install
```

启动Koa服务
```javascript
node server.js

```
或
```javascript
npm run serve

```
